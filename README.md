Simple code examples for [SwiftBond](https://github.com/SwiftBond/Bond)
- Bond v4.3.1
- Swift v2.2.1

## Build
```
$ git clone https://github.com/aminaura/SwiftBondSampler.git
$ pod install
$ open "SwiftBondSampler.xcworkspace"
```
